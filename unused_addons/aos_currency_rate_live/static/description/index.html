Indonesia Currency Rate Live on Rate Bank in Indonesia

Features:

  • Update currency rate manual and automatic
  • Use selected to get rate .
  • BCA, BNI, BRI, CIMB
  • Sign Up at https://klikodoo.id/web/signup to get Key and Secret

  • How to use

Sign up to https://klikodoo.id/web/signup Go to My Account Request a Key
Sign up to https://klikodoo.id/web/signup Go to Add API Name than Click Request
Sign up to https://klikodoo.id/web/signup Go to Click on My KlikAPI
Sign up to https://klikodoo.id/web/signup Go to Your Key and Secret here
Sign up to https://klikodoo.id/web/signup Go to Add your Key and Secret on General Settings
Select Bank Provider. Try to manual get currency rate
Rate Currency updated.
